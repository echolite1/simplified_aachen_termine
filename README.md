# aachen_termine_bot

Simple Telegram bot, that helps you book an appointment at Stadt Aachen.

How to start:
- clone it
- run ``npm i``
- create .env folder: touch .env > BOT_TOKEN=00000000:a0sf0as0fsd0fa0fs
- add ur apikey and token to ``.env``
- run ``npm start``
- to stop execution press ``control+C``
- enjoy!
